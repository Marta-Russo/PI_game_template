loop

()

Inherited from Base but overwritten in app/components/exp-lookit-games/discreteBounce.js:152

Main loop of the game. Set initial position of the ball in a box and starting sound . After that start ball trajectory. If ball hits the target or missed the target (hits any screen edge) wait util user places the paddle to starting position and move the ball to initial position. Increase the score if ball hits the target. Currently game has 2 states when ball hits the target : good (hits the target within the window), very good (hits exact target)
