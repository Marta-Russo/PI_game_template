ExpFrameGamesComponent Class

Extends VideoRecord
Defined in: app/components/exp-lookit-games/component.js:13
Collection: exp-player

Frame to implement various games interventions. The games are developed as a separate HTML5 canvas modules.

Current implementation of template has 3 scenarios to display :

  • Intro page (shows the initial game image with door button)
  • Progress page (shows the progress page with slot machines with dynamic button )
  • Game instructions page (shows the game instructions set as template parameters)

After each game, get the response that game is finished, save the data (pass the data to the appropriate game object array) and proceed to the next game/step.

Current implementation has 5 game types :

  • 0: Light the night sky
  • 1: Monster slime
  • 2: Catch the rat
  • 3: Space mechanic
  • 4: Brake the wall
