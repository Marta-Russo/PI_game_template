Main loop of the game Set initial position of the ball in a box and starting sound . After that start ball trajectory. If ball hits the target or missed the target(window) show the ball in the window and selected window clicked by user (indicate the window background with color). Increase the score if ball hits the window. Move the ball to initial position. Wait for some time until rattle sound played.
